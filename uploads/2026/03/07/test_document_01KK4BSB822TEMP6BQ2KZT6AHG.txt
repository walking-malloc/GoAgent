Ragent 文档处理系统测试文档

这是一个简单的文本测试文档，用于验证文档上传和解析功能。

系统功能：
1. 文档上传：支持 PDF、Word、Markdown、文本等多种格式
2. 文档解析：自动提取文档中的文本内容
3. 智能分块：将长文档分割成合适大小的文本块
4. 向量化：将文本转换为向量表示，存储到 Milvus
5. 语义检索：基于向量相似度进行文档检索

技术特点：
- 使用 Go 语言开发，性能优异
- 支持异步处理，不阻塞用户操作
- 自动处理文档解析和向量化流程
- 提供完整的 RESTful API 接口

测试内容：

系统应该能够正确处理中文字符，包括标点符号和特殊字符。
